# 14_Integrations

## Overview

This domain covers third-party integrations, external systems, and integration patterns for ACSE.

## Documentation

This directory contains integration patterns, connector specifications, and integration guides.


This begins the enterprise integration architecture and governance specifications, covering how ACSE securely integrates with external systems, SaaS platforms, identity providers, AI providers, event buses, APIs, and enterprise data sources.

The planned sequence is:

| INT-001 | Integration Strategy & Architecture             |

| INT-003 | Event-Driven Integration Strategy               |
