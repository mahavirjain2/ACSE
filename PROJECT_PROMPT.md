# ACSE Project Prompt

## Project Overview

ACSE is a comprehensive framework designed to guide organizations in building scalable, secure, and compliant applications. This project provides structured documentation, best practices, and guidelines across all major engineering domains.

## Project Goals

1. **Establish Standards**: Create consistent standards across product, engineering, and operations
2. **Enable Scalability**: Provide frameworks that support growth from startup to enterprise
3. **Ensure Security**: Embed security practices throughout the development lifecycle
4. **Achieve Compliance**: Simplify compliance with regulatory requirements
5. **Foster Collaboration**: Create a collaborative environment for sharing knowledge

## Key Principles

- **Comprehensive**: Cover all aspects of application development
- **Modular**: Allow organizations to adopt practices incrementally
- **Practical**: Provide actionable guidance, not just theory
- **Evolving**: Continuously update based on industry trends
- **Inclusive**: Welcome diverse perspectives and contributions

## Documentation Domains

The project is organized into 19 domains, each addressing specific aspects:

1. **00_Foundations** - Fundamental concepts
2. **01_Product** - Product strategy and management
3. **02_Business** - Business processes
4. **03_Architecture** - System architecture
5. **04_Data** - Data management
6. **05_API** - API development
7. **06_UI_UX** - User interface and experience
8. **07_AI** - AI and machine learning
9. **08_Compliance** - Regulatory compliance
10. **09_Security** - Security practices
11. **10_Engineering** - Engineering standards
12. **11_DevOps** - Deployment and operations
13. **12_Testing** - Testing strategies
14. **13_Operations** - Operational procedures
15. **14_Integrations** - Third-party integrations
16. **15_Governance** - Governance structures
17. **16_Reference** - Reference materials
18. **17_Templates** - Reusable templates
19. **18_Assets** - Design assets

## Success Criteria

- All documentation domains populated with core content
- Community engagement and contributions
- Adoption by organizations
- Regular updates and maintenance
- Positive feedback from users
